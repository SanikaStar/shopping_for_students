**Online Shopping System (C++ Project)**
This is a C++ console-based online shopping application.
It simulates a small shopping system where a customer can browse multiple product categories,
select items, enter quantities, and get a bill. Customer details and purchase information are saved to a file.

**üí° Features**
Customer Details:
Enter first name, middle name, and surname.
Input validation ensures names contain only letters.
**Product Categories:**
Bags
Keyboards
Mouse
Laptops
Headphones
Spectacles

**Billing System:**
Calculates bill for each product based on quantity.
Displays current bill after each purchase.
Generates total bill at the end of shopping.

**File Handling:**
Customer data and bill details are saved in file.txt.
Supports appending data for multiple shopping sessions.

**User Interaction:**

Option to continue shopping multiple times.

Input validation for choices and quantity.

 **How to Run**
Method 1: Using VS Code Terminal (Windows)
Open the project folder in VS Code.
Open Terminal (`Ctrl + ``).

**Compile the program:**

g++ OnlineShopping.cpp -o OnlineShopping.exe

**Run the program:**

.\OnlineShopping.exe

**Method 2: Using Command Line**
Open Command Prompt and navigate to the project folder:
cd path\to\shopping_for_students-main

**Compile:**
g++ OnlineShopping.cpp -o OnlineShopping.exe
**Run:**
OnlineShopping.exe
**Usage**
Enter customer details (first name, middle name, surname).
Enter S or s to start shopping.
Select a product category and item number.
Enter the quantity.
Repeat shopping if desired.
When finished, view the total bill and check file.txt for saved data.
